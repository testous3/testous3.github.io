Politique de confidentialité – Ryven

Dernière mise à jour : Avril 2026

1. Introduction

L’application Ryven respecte votre vie privée et s’engage à protéger les données personnelles de ses utilisateurs. Cette politique explique quelles informations sont collectées, comment elles sont utilisées et vos droits.

2. Données collectées

Ryven collecte uniquement les données nécessaires au fonctionnement de l’application.

Données d’utilisation

  • Informations liées aux courses analysées (adresses, estimations)
  • Actions effectuées dans l’application

Données issues de l’analyse d’écran (OCR)

  • Texte affiché à l’écran (ex : détails d’une course)
  • Utilisé uniquement pour extraire les informations nécessaires (adresses, prix, etc.)

Informations techniques

  • Modèle de téléphone
  • Version Android
  • Logs d’erreurs (crash)

3. Données non collectées

  • Pas de données de localisation GPS
  • Pas de données personnelles sensibles
  • Pas de données de paiement

4. Utilisation des données

  • Fournir les fonctionnalités principales de Ryven
  • Analyser les informations de course
  • Améliorer la performance et la stabilité

5. Partage des données

Ryven ne vend pas vos données personnelles.

Certaines données peuvent être transmises à des services tiers uniquement pour le bon fonctionnement de l’application :

  • Google (services techniques)
  • Mapbox ou TomTom (calculs de trajets)

6. Stockage des données

  • Les données sont principalement traitées localement sur votre appareil
  • Aucune donnée personnelle n’est stockée de manière permanente sur des serveurs externes

7. Sécurité

Nous mettons en œuvre des mesures techniques pour protéger vos données contre tout accès non autorisé.

8. Vos droits

Conformément au RGPD, vous pouvez :

  • Demander l’accès à vos données
  • Demander leur suppression
  • Retirer votre consentement

Contact : contact@ryven.app

9. Permissions

  • Accès écran / OCR : pour analyser les informations de course
  • Internet : pour les services de calcul

10. Modifications

Cette politique peut être mise à jour à tout moment.
